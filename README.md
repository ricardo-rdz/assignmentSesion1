# Frontend Automation Challenge — TestCafe

Automated end-to-end tests for a frontend web application using **TestCafe** and the **Page Object Model** pattern.

## 🧪 What it tests

- UI interactions and navigation flows
- Form validation and user input
- Element visibility and state assertions

## 🚀 Getting Started

**Prerequisites:** Node.js 14+

```bash
# Install dependencies
npm install

# Run tests
npx testcafe chrome page_object_model/
```
